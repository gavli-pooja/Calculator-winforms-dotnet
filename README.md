# Calucalor

Simple Windows Calculator with modern **dark theme**.

## Features
- Basic operations: +, -, ×, ÷
- Percentage (%)
- Sign flip (+/-)
- Clear (AC)
- Decimal support
- Responsive button hovers

## How to Run
1. Open terminal in project folder
2. ```
   cd Calucalor
   dotnet run
   ```

## Build
```
dotnet build
```

Built with .NET  WinForms.
